# Donezo Dashboard

A React task management dashboard built for the Frontend Intern assignment.

## Tech Stack
- React + Vite
- Tailwind CSS
- Recharts (charts)
- Framer Motion (animations)
- React Router v7
- Axios (API + JWT interceptors)

## Features
- JWT login with persistent auth
- Private route protection
- Live data from REST API
- Animated dashboard cards
- Fully responsive layout

## Setup
Write VITE_API_BASE_URL in .env then
npm install
npm run dev

## API
Base URL: https://task-api-eight-flax.vercel.app
Login: POST /api/login
